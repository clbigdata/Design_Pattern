# Design_Pattern
Design Pattern code description 设计模式代码示例

## 1.创建型设计模式
> 这类模式提供创建对象的机制， 能够提升已有代码的灵活性和可复用性。
- 工厂模式
- 抽象工厂模式
- 原型模式
- 生成器模式
- 单例模式
## 2.结构型设计模式
> 这类模式介绍如何将对象和类组装成较大的结构， 并同时保持结构的灵活和高效。
- 适配器模式
- 桥接模式
- 组合模式
- 装饰模式
- 外观模式
- 享元模式
- 代理模式
## 3.行为型设计模式
> 类模式负责对象间的高效沟通和职责委派。
- 责任链模式
- 命令模式
- 迭代器模式
- 中介者模式
- 观察者模式
- 备忘录模式
- 观察者模式
- 状态模式
- 策略模式
- 模板方法模式
- 访问者模式
